## Taco Loco

is a URL shortener


#### Requirements

- [PHP](http://php.net/) >= 5.4.0
- [Tipsy](http://tipsy.la/) >= 0.7.1
- [Composer](https://getcomposer.org/)

#### Installation

1. Install Tipsy and the autoload library: **composer install**
2. Import the database schema in **db/schema.sql**
